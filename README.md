\# 🎓 College Online Voting System (Backend)



\## 📌 Description



This is the backend of a college-level online voting system developed using \*\*PHP and MySQL\*\*.



\## ⚙️ Features



\* Admin registration \& login

\* Student login system

\* Candidate application \& approval

\* Election creation and management

\* Secure voting system

\* Result publishing



\## 🛠 Technologies Used



\* PHP

\* MySQL (XAMPP)

\* REST API



\## 📂 Folder Structure



\* `api/` → API endpoints

\* `config/` → Database configuration

\* `middleware/` → Authentication \& validation

\* `sql/` → Database schema

\* `utils/` → Helper functions



\## 🚀 How to Run



1\. Start XAMPP (Apache + MySQL)

2\. Import `sql/schema.sql`

3\. Place project in `htdocs`

4\. Access via browser



\## 🔐 Security



\* Password hashing

\* Vote privacy handling
